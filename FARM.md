# 种植系统

+ 类似星露谷种植系统的游戏

+ 基于phaser进行开发

+ 构建货币系统

货币获取方式包括签到，日记，售卖作物等

+ 作物
    - 随时间长大
    - 浇水减少成熟的时间
    - 需要用货币购买种子
    - 需要情侣系统达到一定程度才能解锁特定植物，植物应该具有不同的特性。
    - 在app/static/images/crops中提供了部分作物的素材，开发应具备易扩展性
    - 作物不同的生长阶段应该具有不用的形态。

+ 游戏方式
    - 使用app/static/images/carlorine提供的素材构建精灵，使用点击的方式操控精灵移动，不要按照方格的方式，应该能够自由移动的效果。
    - 地图构建可以使用app/static/images/ground提供的部分素材。
    - 应当能够切换工具，例如切换到水壶时才能浇水，切换到锄头时锄地，切换到种子时种植
    - 构建水塘等水域，具备钓鱼的功能，钓鱼也能售卖也能获取货币
    - 缺少的素材可以用文字暂时顶替。或者使用其他方式暂代，后续进行补充。
